#**SISTEMA DE GERENCIAMENTO DE FESTA - Trabalho Estrutura de Dados 03/2026**#

##PROPOSTA:
###Proposta é desenvolver um sistema de cadastro de dados de uma festa. Incluindo convidados, trabalhadores e convites. Realizar listagem de convidados, confirmação dos mesmos, registro de trabalhadores, cálculo da conta da festa e operação dos registros.

##TÉCNICA:
###A parte técnica consistia em desenvolver o sistema na linguagem C, para solidificar o uso de ponteiros e alocação dinâmica e estática de memória. Assim também, como executar integração de structs e suas relações.

##RELATÓRIO:
###Foi um trabalho importante para compreender de fato como funciona alocação de memória dinâmica e estática. Permitiu um exercício prática do uso de ponteiros, assim como o desenvolvimento de um CRUD simples. Busquei compreender todos os erros de terminal, evitando o máximo qualquer auxílio. Ao fim do trabalho, após constantes revisões, lanceis breves questionamentos a IA para compreender boas práticas e especificidades que não compreendia (ex: a função memset() que não conhecia).
